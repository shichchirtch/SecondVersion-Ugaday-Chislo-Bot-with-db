# Бот - Угадай число

## Инфо

Телеграм бот который загадывает число. Цель игрока отгадать это число. Бот создавался для обучающих целей.

## Как запустить

Чтобы запустить

```bash
docker compose up -d
```

Посмотреть логи

```bash
docker compose logs -f
```

Выключить бота

```bash
docker compose down
```

## Используемые технологии

- docker, docker compose
- PostgreSQL
- aiogram
